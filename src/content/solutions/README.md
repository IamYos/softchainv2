# Solutions Content

These markdown files are the source copy for Softchain's four approved solution pillars. The pillar titles must stay aligned with the homepage:

- Software Design & Engineering
- AI Systems
- IT Infrastructure
- Technology Management

Each solution file contains:

- A short website version written for the solutions page.
- Two divided website sections, so the animated solutions graphic can present each pillar in focused parts.
- An extended company profile version for brochures, proposals, and long-form business material.

Sub-section titles can change as the design evolves, but they should always sit under one of the four approved solution pillars instead of becoming new top-level solutions.
